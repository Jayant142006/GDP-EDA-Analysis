# Global GDP Exploratory Data Analysis

## 📌 Project Overview
This project focuses on **Exploratory Data Analysis (EDA)** of global GDP data to uncover patterns, trends, and insights about the economic distribution of countries.

The dataset used in this project was prepared from my previous project:
**GDP Data Cleaning with Pandas**, where the raw GDP data was cleaned and transformed into a usable format.

This project demonstrates how Python can be used not just for cleaning data, but also for **analyzing and visualizing it to extract meaningful insights**.

---

## 🎯 Objective
The main objective of this project is to:

- Analyze GDP values of countries
- Identify top and bottom economies
- Understand GDP distribution across countries
- Derive meaningful insights from visual analysis

---

## 🛠 Tools & Libraries Used
- **Python**
- **Pandas**
- **NumPy**
- **Matplotlib**
- **Seaborn**
- **Jupyter Notebook**
